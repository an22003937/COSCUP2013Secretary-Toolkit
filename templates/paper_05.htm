焦點議程

Open Data Initiatives for Taiwan

近年全球興起一浪開放資料 (Open Data) 熱潮，民眾從開放資料中得到便利，也可以從中了解到國家和城市的情況。政府對開放資料支持、態度和行動，對國家和城市開放資料的發展速度有一定影響。台灣 在 2012 年尾正式推動 open data，政府部門在 2013 年第二季開始釋出一些數據。

今年 COSCUP 有幸邀請到行政院政務委員 - 張善政委員，在第一天的 keynote 介紹台灣政府在開放資料上的進展。

大家記得準時到場啊！！！

時間\地點：8/3 09:10 - 09:40 \ 101ABCD (全場地聯播)

Open Source Engine powering Big Data in the Cloud

在IT產業14年的 Markku 目前在AWS擔任 APAC Technology Evangelist ，Markku 將透過他在芬蘭、中國大陸、新加坡的工作經驗和大家解釋 Big Data 是如何和開源工具、雲端計算 一起運作的， 透過實際案例告訴各位 Data Driven Businesses 會碰到的挑戰、 機會以及實作上的技巧。

時間\地點：8/3 10:15~11:15 \ 101會議室，全場聯播

From Git to Production - Amazon Web Services in Action

Markku會在這次的議程中我們將在現場 demo 如何使用 ElasticBeanstalk 來做到自動部署, load balancing, auto scaling , 以及伺服器狀態監控。

時間\地點：8/3 11:45~12:15 \ 103會議室

Raspberry PI and the future – Leveraging Open Source hardware and software for education and commercial development

Raspberry Pi 是 Broadcom 在開源軟硬體界的 SOC 解決方案首次嘗試，這次的簡報中我們將介紹 Raspberry Pi 基金會以及 Broadcom 在參與的過程中面臨的挑戰，又是如何將這個產品帶到市場上的，最後再跟大家介紹今後的計畫以及一些預定導入的新玩意兒。這場 keynote 也會頒出RS Components 活動送的 10塊 Raspberry Pi !

時間\地點：8/4 15:10~16:10 \ 101 會議室 全場聯播

DesignSpark 免費設計資源與 Raspberry Pi 新應用

知道Raspberry Pi推出一年多後有什麼改進和新應用嗎? DesignSpark 的技術演講將告訴你如何使用 PCB 設計軟體，及數萬個符號與 3D 模型來玩 Raspberry Pi!

時間\地點：8/4 14:00~14:30 \ 102 會議室

純自由軟體的虛 擬偶像歌手實作

開源軟件不只是我們的眼睛從顯示器中看到，也可以透過我們的耳朵聽得見。

自由軟件信眾 Paul Liu (Ying-Chun Liu) 一直想向專屬軟件下戰書。Paul 會在 COSCUP 上介紹目前華語偶像歌手實作進度，自動處理子音與母音切割的技術，語音合成的腳本，並會示範目前所擁有的成果。

時間\地點：8/3 11:45 - 12:15 \ 201D
